BSc AI Study Assistant

Project Goal

Build a Production-Grade RAG (Retrieval-Augmented Generation) System that answers questions from semester notes and study materials using Hybrid Retrieval, Reranking, Citations, Monitoring, Evaluation, and CI Pipeline.

---

Tech Stack

- Python
- FastAPI
- LangChain
- ChromaDB
- OpenAI
- Sentence Transformers
- BM25
- RAGAS
- Langfuse
- GitHub Actions

---

Features

- PDF-based Question Answering
- Hybrid Retrieval (BM25 + Vector Search)
- Cross-Encoder Reranking
- Citation-based Responses
- Monitoring & Observability
- Latency Tracking (p50 / p95)
- Cost Per Request
- Quality Evaluation using RAGAS
- CI Regression Gating
- Deployment Ready
