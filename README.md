# BSc AI Study Assistant

## Project Goal

Build a Production-Grade RAG (Retrieval-Augmented Generation) System that answers questions from semester notes and study materials using Hybrid Retrieval, Monitoring, Evaluation, and Local LLM Integration.

---

## Tech Stack

* Python
* Streamlit
* LangChain
* ChromaDB
* HuggingFace Embeddings
* BM25
* Ollama
* Llama 3
* Sentence Transformers
* GitHub

---

## Features

* PDF-based Question Answering
* Hybrid Retrieval (BM25 + Vector Search)
* Local LLM Integration using Ollama + Llama 3
* Monitoring & Observability
* Response Time Tracking
* Retrieved Document Logging
* Answer Quality Evaluation Pipeline
* Streamlit Frontend Chatbot UI
* GitHub Project Showcase

---

BSc AI Semester Notes Assistant
